# Development Commands

## Start Development Server
npm run dev

## Build for Production
npm run build

## Start Production Server
npm start

## Run Linter
npm run lint

## Clean Build
Remove-Item -Recurse -Force .next -ErrorAction SilentlyContinue
npm run dev

## Install Dependencies
npm install

## Update Dependencies
npm update

## Check for Errors
npx tsc --noEmit
